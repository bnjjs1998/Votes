Comment initialiser le projet

Dans un premier temps, il faut créer impérativement un compte avec le pseudo "admin",
le mot de passe, c'est vous qui le choisissez.

Si le pseudo est différent "admin", vous aurez le rôle "utilisateur".

git clone https://github.com/bnjjs1998/Votes.git
cd project

python -m venv env
source env/bin/activate  # macOS/Linux
env\Scripts\activate     # Windows

pip install -r requirements.txt

Créer une base de données MongoDB
